斜屋犯罪

发表于 2019-11-28 | 分类于 读书笔记

很久没有读完一本书了。上一本没读完的书是《悲惨世界》，在零散的空闲时间啃了百分之三十，终于还是放弃了。
读读书，烧烧脑，可以让大脑处于活跃状态。出于这个目的，推理小说是很好的阅读材料。而《斜屋犯罪》这本书，是我很爱的公众号作为入门级推理小说推荐的。存了那份书单，放弃《悲惨世界》后立马亚马逊下单。
故事以一个聚会和两个“密室”凶案为背景，一点点展开。虽然在小说最前面已经介绍过每个人物，但具体各个人物之间的牵扯是在后文才看得到的。和现实很像不是吗？表面看来大家都是朋友，实际上，谁知道呢。
这本书需要一些空间想象能力。而我，从初中开始解立体几何问题都喜欢用计算方法，我难以想象空间，不擅长使用几何法。这本书也是一样。解谜的关键其实在于这个建筑，感兴趣的朋友第一次读的时候就多多多多多多留心下。
我本来觉得，推理小说，就是根据已有的信息推测出答案。但不是的，它不是数学题。在数学题里，所有解题的条件都给了我们；而现实中的谜题，条件在哪需要自己寻找，也需要自己去甄别哪些是冗余条件。所以说现实比数学难的多多了。
但还挺有意思的不是吗？情理之中。再读一遍时，谜题的答案有迹可循，有好多疑点初读时都没留意。是不是意味着，初读时我只是跟着作者读，没有自己的主见呢？自己参与会更有劲吧！

申请季焦虑的碎碎念
